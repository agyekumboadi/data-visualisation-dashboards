# Excel Dashboards

This folder contains Excel dashboard exports designed to demonstrate:
- KPI tiles and summary cards
- dealer-level comparisons
- model-level comparisons
- time-based profit patterns
- dashboard layout and formatting

These visuals are commonly built using combinations of:
- PivotTables / PivotCharts
- slicers (filters)
- calculated fields
- consistent formatting (titles, spacing, grid alignment)

---

## Dashboard: Car Model Sales Dashboard

**File**
- `Excel_Dashboard_CarModel_Sales.jpeg`

**What it shows**
A summary layout that typically includes:
- KPI totals (Profit, Quantity Sold, Average Quantity Sold)
- Quantity Sold by Model (bar chart)
- Profit by Dealer ID (bar chart)

**How to interpret**
- Quantity Sold by Model: identifies demand leaders.
- Profit by Dealer ID: identifies channel value contribution.
- KPIs: provide the overall baseline for the period.

**Why it’s useful**
- Combines performance + distribution in one view.
- Helps answer:
  - which models sell most
  - which dealers generate most profit
  - overall profit/volume levels

## Visual: Profit by Date & Model

**File**
- `Excel_Profit_by_Date_&_Model.jpeg`

**What it shows**
A time-based profit breakdown split by model.

**How to interpret**
- Compare bars (or series) to see:
  - which model is most profitable
  - whether profit rises/falls at specific periods
- Useful for identifying:
  - seasonal effects
  - unusual spikes (potential campaigns or pricing changes)

**Recommended usage**
- Add slicers for:
  - Year
  - Dealer ID
  - Model
to quickly isolate drivers.

## Visual: Profit by Year

**File**
- `Excel_Profit_by_Year.jpeg`

**What it shows**
Year-over-year profit comparison.

**How to interpret**
- Upward movement indicates improving profitability.
- Declines suggest performance drops or cost/price pressures.

**Why it’s useful**
- Executive-level indicator: quickly shows long-term direction.
- Helps support annual reporting and planning.

---

## Visual: Quantity Sold by Dealer ID

**File**
- `Excel_Quantity_Sold_by_DealerID.jpeg`

**What it shows**
Dealer-level sales volume comparison.

**How to interpret**
- High volume dealer ≠ high profit dealer (not always).
- This chart is best used together with profit charts.

**Why it’s useful**
- Identifies key distribution channels.
- Helps decide where to focus support, marketing, or supply.
